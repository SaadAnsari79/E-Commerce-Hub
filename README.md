# EcommerceHub

Spring Boot E-commerce Backend using JPA, PostgreSQL, Spring Security and JWT Authentication.

## Tech Stack

* Java
* Spring Boot
* Spring Data JPA
* Spring Security
* JWT Authentication
* PostgreSQL
* Maven

## Features

* User Authentication & Authorization
* Role-Based Access Control
* Product Management
* Category Management
* Shopping Cart
* Order Processing
* REST APIs
* Database Integration using JPA/Hibernate

## Architecture

* Controller Layer
* Service Layer
* Repository Layer
* DTO Pattern
* Exception Handling
* Validation
